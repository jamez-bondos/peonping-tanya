# OpenPeon Tanya (Red Alert 2)

This repository contains a sound pack for the [Open Peon initiative](https://openpeon.com/).

It provides **Command & Conquer: Red Alert 2** voice lines for **Tanya**, the iconic Allied commando.

## Contents

- `openpeon.json`: package metadata and sound mapping
- `sounds/`: MP3 files used by the pack

## Package Details

- Name: `tanya_ra2`
- Display name: `Tanya (Red Alert 2)`
- Language: `en`
- Version: `1.0.0`
- License: `CC-BY-NC-4.0`

## About Open Peon

Open Peon is an initiative for reusable voice/sound packs for developer workflows.

Learn more at: <https://openpeon.com/>
